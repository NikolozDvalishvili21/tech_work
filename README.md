**React.js, Tailwind CSS, shadcn/ui, Swiper და anime.js ბიბლიოთეკებით შექმნილი ერთგვერდიანი თანამედროვე web-app.**

**ინსტალაციის ინსტრუქცია**

დაკლონე Repository: git clone https://github.com/NikolozDvalishvili21/tech_work.git

შედი პროექტის საქაღალდეში: cd tech_work

დააინსტალირე საჭირო ბიბლიოთეკები: npm install

გაუშვი პროექტი ლოკალურად: npm run dev

გახსენი ბრაუზერში: http://localhost:5173

**კომპონენტი და აღწერა**

HeroSlider: Swiper-ზე დაფუძნებული ჰერო სექცია autoplay და ტექსტების fade ეფექტით (animejs)
QuickActions: shadcn/ui buttons anime.js ანიმაციით
Cards: shadcn/ui ბარათები შემოსვლის ანიმაციით (fade + pop up საიტის რეფრეშზე)
Footer: Footer მცირე ფერების ცვალებადობის ანიმაციით

**ბიბლიოთეკები და დანიშნულება**

React.js: UI ბიბლიოთეკა, გამოიყენება აპლიკაციის component-based სტრუქტურის შესაქმნელად.
Tailwind CSS: სწრაფი და მოქნილი styling-ისთვის განკუთვნილი utility-first CSS ბიბლიოთეკა.
shadcn/ui: მზა და მინიმალისტური UI კომპონენტების ბიბლიოთეკა (Button, Card).
Swiper.js: გამოიყენება ჰერო სექციაში სლაიდერის შესაქმნელად autoplay და pagination ფუნქციებით.
anime.js: ანიმაციების ბიბლიოთეკა.
Vite: თანამედროვე და სწრაფი development bundler React აპლიკაციის გასაშვებად და ასაწყობად.
